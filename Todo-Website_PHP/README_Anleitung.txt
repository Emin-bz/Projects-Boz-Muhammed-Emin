Dies ist die Anleitung für das Aufsetzen dieser Website.

Um die Website mit der Datenbank benutzen zu können benötigen Sie XAMPP. 


1. Starten sie in XAMPP die Module "Apache" und "MySQL".

2. Danach drücken Sie bei dem Modul "MySQL" auf "Admin" um die Datenbank zu importieren.

3. Sie sind nun auf "phpmyadmin". Drücken Sie links bei der Datenbankliste auf "Neu" um eine Datenbank zu erstellen.

4. Wenn die Datenbank erstellt ist, gehen Sie oben auf "Importieren" und dann auf "Datei auswählen".

5. Wählen Sie jetzt die "todo.sql" Datei aus, welches sich im Verzeichnis "Datenbanken" befindet.

6. Verschieben Sie den Projektordner nun in das Verzeichnis "C:/XAMPP/htdocs/".

6. Jetzt müssen Sie im URL "localhost" + den Pfad angeben und die Website benutzen.
